# クルマログ（kuruma-log）

> 思いついたら即記録。シンプルなメンテナンス管理アプリ

---

## 📱 Live Demo

🚀 **[クルマログを開く](https://shun-devlab.github.io/kuruma-log/)**

---

## 🎯 概要

**クルマログ**は、車のメンテナンス記録を簡単に管理できるアプリです。

- ⛽ ガソリン給油
- 🛢️ オイル交換
- 🛞 タイヤ交換
- 🚿 洗車
- ⚠️ 故障・修理

をホーム画面から **3-5秒で記録**。次回メンテナンスの時期を自動計算して通知します。

---

## ✨ 特徴

### シンプル

ぴよログ並みのシンプル UI。複雑な設定は不要。

### ワンタップ記録

[⛽] [🛢️] [🛞] [🚿] [⚠️] をタップするだけで記録完了。

### 自動計算

次回予定（「あと 1,200km でオイル交換」）を自動計算。

### グラフ表示

月別支出・走行距離推移をグラフで可視化。

### ローカル保存

IndexDB でブラウザにローカル保存。クラウド同期不要。

---

## 🛠️ 技術スタック

- **フロント**: Vue.js 3 + Vite
- **ストレージ**: IndexDB
- **グラフ**: Chart.js
- **デプロイ**: GitHub Pages
