## 🎯 ぴよログから学ぶシンプル設計

### **ぴよログの 特徴**

```
【ホーム画面構成】

┌─────────────────────┐
│ 📅 日付（タイムライン）│ ← シンプル
├─────────────────────┤
│                     │
│   時系列表示        │
│  (0時, 1時...)     │
│                     │
│  各時間帯に記録     │
│  「母乳」「ミルク」 │
│  「寝る」「起きる」 │
│                     │
├─────────────────────┤
│  [📝] [📊] [📈]... │ ← タブナビ（5つ）
└─────────────────────┘

【記録方法】
下部にアイコンがズラッと並ぶ
→ タップするだけで完了
→ 煩雑な入力フォームなし
```

---

## 📐 ぴよログのシンプル戦略

### **戦略 1: 「事前に選択肢を用意」**

```
❌ 悪い例（Drivvo っぽい）:
「メンテナンス内容を入力してください」→ テキストボックス

✅ 良い例（ぴよログ）:
「母乳」「ミルク」「寝る」「起きる」などのアイコンをズラッと表示
→ タップするだけ
```

**クルマログへの適用**:
```
【記録画面】
┌─────────────────────┐
│ ⛽ ガソリン給油     │
│ 🛢️  オイル交換      │
│ 🛞 タイヤ交換       │
│ 🚿 洗車             │
│ ⚠️ 故障・修理       │
└─────────────────────┘
→ アイコンをタップして選択
→ 最小限の入力で完了
```

---

### **戦略 2: 「タイムライン形式」**

```
ぴよログ:
0時 ┃
1時 ┃
2時 ┃ 🍼 (ミルク)
3時 ┃
...
19時 ┃ 😴 (寝た)
20時 ┃
21時 ┃

→ 時系列が一目瞭然
→ 「いつ何をしたか」が視覚的にわかる
```

**クルマログへの適用**:
```
【ダッシュボード（案）】

2026-03-01 (日)
─────────────────────
  ⛽ ガソリン給油 (45,200km)
  💰 ¥3,500
  
2026-02-28 (土)
─────────────────────
  🚿 洗車
  
2026-02-20 (木)
─────────────────────
  ⚠️ エンジン修理
  💰 ¥15,000

→ 新しい順に、記録が並ぶ
→ スクロールで過去が見える
```

---

### **戦略 3: 「下部タブで機能分離」**

```
ぴよログの下部タブ:
[記録] [まとめ] [成長曲線] [予防接種] [子育て支援] [メニュー]

→ 最上位の機能は 5-6 個のみ
→ 複雑な設定はメニューの奥
→ 迷わない
```

**クルマログへの適用**:
```
【下部タブ（案）】
[ホーム] [記録] [統計] [設定]

または

[ホーム] [一覧] [グラフ] [設定]

→ 4 個のタブのみ
→ シンプル
```

---

### **戦略 4: 「クイック記録ボタン」**

```
ぴよログ:
下部に大きなピンク色のボタン:
[👶] [🍼] [😴] ... 

→ 最も使う機能を前に出す
→ タップ 1 つで記録開始
```

**クルマログへの適用**:
```
【ホーム画面のアクション】
┌──────────────────────┐
│ 【次回予定】         │
│ ⭕ オイル交換:      │
│    あと 1,200km      │
├──────────────────────┤
│ 【最近の記録】       │
│ ⛽ 2026-03-01 給油   │
│ 🚿 2026-02-28 洗車   │
├──────────────────────┤
│ 【支出】             │
│ 今月: ¥23,500        │
└──────────────────────┘

┌─────────────────────┐
│   [+ 新規記録]      │ ← 大きなボタン
└─────────────────────┘

→ タップすると:
  ⛽ 給油 / 🛢️ オイル交換 / 🛞 タイヤ / 🚿 洗車 / ⚠️ 修理
  を選択
```

## 🎨 UI デザイン方針（更新版）

### **クルマログの UI は「ぴよログ級」**

```
【特徴】
1️⃣ カラフルだけど落ち着いた色
   （ぴよログのピンクみたいに）
   
2️⃣ アイコン + テキストの組み合わせ
   （文字だけより視認性高い）
   
3️⃣ 下部ナビは 4-5 個のタブのみ
   （複雑なメニューなし）
   
4️⃣ メイン操作は「タップ」だけ
   （テキスト入力は最小限）
   
5️⃣ 時系列で記録が見える
   （ぴよログのタイムライン方式）
```

## ✅ UI デザイン基本方針（確定）

### **クルマログ UI の 5 つのルール**

1. **アイコン + テキスト** → 視認性優先
2. **選択式 > テキスト入力** → タップで完了
3. **下部タブは 4 個以下** → シンプルナビ
4. **メイン機能は上に** → 次回予定を目立たせる
5. **色は落ち着いた車テーマ** → ぴよログのピンクのような「優しい色」

## 🎨 カラースキーム案

```
【クルマログのカラーパレット】

Primary: 淡いカーブルー / グレー
Secondary: 明るいオレンジ（重要情報）
Accent: グリーン（完了・OK）/ レッド（警告）
Background: ホワイト / ライトグレー
Text: ダークグレー

例：
- 【次回予定】: オレンジで目立たせる
- 【完了済み】: グリーンで表示
- 【警告】: レッドで表示
```
