## 🔍 顧客の課題（Problem）

### **課題 1: メンテナンス履歴を覚えられない** ⚠️

**具体的な状況:**
```
「ガソリン給油したのいつだっけ？」
「前回のオイル交換はいつ？」
「今月のガソリン代がいくらかかったか忘れた」
```

**痛み**:
- メンテナンス時期を見落とす
- 記憶に頼っているため間違える
- 手帳・メモが散らばっている
- 後で「確認したい」と思っても遡れない

**頻度**: 毎週（毎回給油時）  
**影響度**: 🟠 中程度（イライラ・不安）

---

### **課題 2: 次のメンテナンスタイミングが分からない** ⏰

**具体的な状況:**
```
「あとどのくらいでオイル交換が必要？」
「タイヤはいつ交換するべき？」
「走行距離が○○km に達したけど、何をすればいい？」
```

**痛み**:
- ディーラーに「頼み」のタイミングがわからない
- 早すぎると無駄（費用がかかる）
- 遅すぎるとエンジンが傷む（後悔）
- 推測で進めて不安

**頻度**: 月 1-2 回（メンテナンス前に毎回）  
**影響度**: 🟠 中程度（判断の迷い・不安）

---

### **課題 3: ガソリン代・修理費の把握ができない** 💰

**具体的な状況:**
```
「今月のガソリン代がいくらかかってるか分からない」
「先月修理に出したのいくら払ったっけ？」
「年間でいくら車にお金がかかってるのか把握できない」
```

**痛み**:
- 家計管理ができない
- 予算立てができない
- 「高い」か「安い」かの判断がない
- 支出パターンが見えない

**頻度**: 月 1 回（家計管理時）  
**影響度**: 🟡 低-中程度（家計管理の効率化）

---

### **課題 4: 複数のツール・メモが散乱している** 📝

**具体的な状況:**
```
「ガソリン代は家計簿アプリ」
「オイル交換は手帳」
「修理記録はレシート」
→ どこに何があるか分からない
```

**痛み**:
- データが分散している
- 一元管理ができない
- 後で確認したいのに見つからない
- 統一されていないのでストレス

**頻度**: 週 1 回（必要な情報を探す時）  
**影響度**: 🟡 低-中程度（管理の手間）

---

## ✅ 解決策（Solution）

### **解決策 1: 一元化されたメンテナンス記録アプリ**

**実装**:
- 3 タップで「ガソリン給油」「オイル交換」などを記録
- 記録は全て 1 つのアプリに保存
- 過去の全記録がいつでも見返せる

**効果**:
- 記憶の負担が減る
- 複数ツール・メモを管理する手間がゼロ
- 過去の傾向が見える（「あのときいくらだった」が瞬時）

---

### **解決策 2: 自動計算される「次回予定」機能**

**実装**:
```
【次回予定】
- オイル交換: あと 1,200km（2026-04-15 予想）
- タイヤ交換: あと 12,000km（2026-06-01 予想）
- 洗車: 前回から 30 日経過（推奨実施）
```

**計算ロジック**:
- オイル交換: 最後の記録から 5,000km or 6 ヶ月
- タイヤ交換: 走行距離ベースで推定
- 洗車: 月 1 回推奨

**効果**:
- 「いつ何をするべき」が一目瞭然
- ディーラーに予約するタイミングが明確
- 不安がなくなる

---

### **解決策 3: 支出の可視化（グラフ・統計）**

**実装**:
```
【月別支出グラフ】
3月: ¥23,500 (ガソリン ¥15,000 + メンテ ¥8,500)

【燃費グラフ】
今月: 8.5km/L（先月比 -0.5km/L）

【年間累計】
総支出: ¥542,000
```

**効果**:
- 「あ、今月は修理費が多かった」と気づく
- 年間予算が立てやすい
- 燃費低下に気づいて早期対応可能

---

### **解決策 4: シンプル・ワンストップのUI**

**実装**:
- ホーム画面で「次回予定」「最近の記録」「支出」が全て見える
- 新規記録は [+] ボタン 1 タップ
- 複数ツールを行き来する必要なし

**効果**:
- 使う手間が減る
- 続かないアプリ（複雑・わかりにくい）ではなく、毎週使い続けられる

## 🎯 価値提案（Value Proposition）

### **コア価値**

```
「車のメンテナンス を全部1つのアプリで管理して、
  次に何をするべきかが 一目瞭然 になるから、
  不安がなくなって 家計が見える。」
```

### **3 つの主な利益**

| 利益 | 説明 | 測定 |
|------|------|------|
| **心理的安心感** | メンテナンスを忘れない | 「あと○km で交換」と可視化 |
| **家計管理の効率化** | ガソリン代・修理費が把握できる | 月別・年間支出グラフ |
| **手間の削減** | 複数ツール・メモから卒業 | 1 つのアプリで全て完結 |

## 🔄 ジャーニーマップ：現在 vs 将来

### **現在（Before）**

```
【日常】
火曜日: ガソリン給油 → 領収書をバッグに入れる
       「あ、今月いくら使ってるか数えなきゃ」

金曜日: 「そういえば前回オイル交換いつだった？」
       手帳を確認 → 「3 ヶ月前か。今月交換しなきゃ」

月末: 家計簿をつける → ガソリン代の領収書を集める
      「あ、この修理代いくらだった？」

痛み: 分散・手間・不安 ⚠️
```

### **将来（After）クルマログ導入**

```
【日常】
火曜日: ガソリン給油 → アプリで 3 タップ「給油完了」
       自動で先月の給油代が表示される「参考」

金曜日: ホーム画面を開く → 「オイル交換：あと 1,200km」
       一目瞭然。ディーラーに電話して予約

月末: アプリの「統計」タブ開く → 月別グラフで支出確認
      「ああ、今月は修理が多かったな」と納得

利益: 一元化・確実・安心 ✅
```
