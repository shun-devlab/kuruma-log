### **03_CUSTOMER_VALUE.md（VCP 分析）**

**内容**: 顧客課題と解決策の詳細分析

**4つの顧客課題**:
1. メンテナンス履歴を覚えられない
2. 次のメンテナンスタイミングが分からない
3. ガソリン代・修理費の把握ができない
4. ツール・メモが散乱している

**対応する 4つの解決策**:
1. 一元化されたメンテナンス記録アプリ
2. 自動計算される「次回予定」機能
3. 支出の可視化（グラフ・統計）
4. シンプル・ワンストップの UI

## 🎯 仕様確定事項（確定版）

### **1. プロダクト仕様**

- ✅ **コア機能**: 5 種類のメンテナンス記録（ガソリン、オイル、タイヤ、洗車、修理）
- ✅ **次回予定**: 自動計算（オイル交換は 5,000km or 6 ヶ月）

- ✅ **ウィジェット**: iOS（Phase 2）、Android（Phase 1）
