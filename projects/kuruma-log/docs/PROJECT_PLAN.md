## 📌 プロジェクト目標

### 主要ゴール
1. ✅ 個人向けカーメンテナンス管理アプリの MVP 完成
2. ✅ 以下の機能を実装：
   - メンテナンス記録（5 種類）
   - 次回予定自動計算
   - 支出分析ダッシュボード
   - シンプルな Web UI
3. ✅ 本人（しゅんさん）が毎週利用できる状態

### 成功判定基準（Definition of Done）
- **機能**: 全 MVP 機能が実装・テスト済み
- **品質**: バグなし、または Low 優先度のみ
- **UX**: 記録を 20 秒以内で完了可能
- **パフォーマンス**: API レスポンス < 500ms

---

## 📊 スコープ管理

### MVP に含まれるもの ✅
- 5 種類のメンテナンス記録機能
- 記録の CRUD 操作（作成・読取・更新・削除）
- 記録一覧表示（フィルタ・ソート）
- 次回予定の自動計算と表示
- 月別支出グラフ
- 走行距離推移グラフ
- Web UI（Vue.js）
- REST API（Node.js + Express）
- SQLite ローカルDB

### MVP に含まれないもの ❌
- スマートフォンアプリ化（iOS/Android）
- クラウド同期・複数デバイス対応
- プッシュ通知
- ユーザー認証・複数ユーザー管理
- 複数台車管理
- CSV エクスポート
- OBD-II 連携
