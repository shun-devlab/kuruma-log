```
【優先順位】
1️⃣ 使いやすさ（機能性）
2️⃣ 見やすさ（可読性）
3️⃣ 美しさ（エレガンス）

機能 > デザイン
わかりやすさ > 洗練さ

- 25-50 代

- スマートフォン経験あり
- 複雑なUIは避けたい層
```

## 📐 タイポグラフィ

### **フォント選定**

```
【日本語】
- 本文: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif
  （システムフォント = シンプル、読みやすい）
- 見出し: 同上（太字でメリハリ）

【数字・英数】
- 走行距離・金額: 'Courier New' or 'Monaco'
  （モノスペース = 数値が読みやすい）

【理由】
- カスタムフォント不要（シンプル）
- 読み込み速度が速い
- 全デバイス対応
```

### **サイズ・ウェイト**

```
【見出し】
h1: 28px / 700（bold）/ 行高 1.3

【セクションタイトル】
h2: 18px / 600（semibold） / 行高 1.4

【本文】
body: 16px / 400（regular） / 行高 1.6
小さいテキスト: 14px / 400 / 行高 1.5

【ボタンテキスト】
16px / 600（semibold）/ 行高 1.4

【ルール】
- 小さい文字は避ける（12px 以下は使わない）
- 高齢層でも読める大きさ
```

## 🎯 記録パネル（詳細）

### **展開時のアニメーション**

```
【タイムライン表示状態】
通常のホーム画面

【⛽ をタップ】
    ↓
【スライドアップ】
- 背景がグレーの半透明オーバーレイ
- パネルが下から上へスライド（300ms）
- アニメーション: cubic-bezier(0, 0, 0.3, 1)

【展開後】
┌────────────────────────┐
│ ⛽ ガソリン給油        │← 上部（固定）
├────────────────────────┤
│ 日付: 2026-03-01       │← 自動入力
│ 走行距離: 45200 km     │← 提案値
│ 金額: [         ] 円   │← 入力欄
│ メモ: [         ]     │← 入力欄
│                        │
│ [保存] [キャンセル]    │
└────────────────────────┘
```

### **フォーム要素の配置**

```
【ガソリン給油の場合】
- 日付: 必須（自動で「今日」）
- 走行距離: 必須（前回から自動提案）
- 金額: オプション
- メモ: オプション

【洗車の場合】
- 日付: 必須（自動で「今日」）
- メモ: オプション
- ※ 走行距離・金額は非表示

【ルール】
- 必須項目: ボールドでラベル表示
- オプション項目: グレーで「（オプション）」と表示
```

## ♿ アクセシビリティ

```

- ボタン最小: 44px × 44px（iPhone 推奨）

【アニメーション】

- prefers-reduced-motion に対応
```
